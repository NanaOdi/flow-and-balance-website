# Flow & Balance

Eine moderne, ruhige und minimalistische Portfolio-Website für eine Pilates- und Yoga-Trainerin.

## Vorschau

Die Website ist als statisches Frontend-Projekt aufgebaut und eignet sich für:

- Portfolio
- GitHub Pages Hosting
- Business-Landingpage
- Design-Demo für Trainerinnen, Coaches oder Wellness-Angebote

## Verwendete Technologien

- HTML5
- CSS3
- JavaScript

## Funktionen

- responsives Layout
- mobile Navigation
- moderne Hero-Section
- Über-mich-Bereich
- Kursübersicht
- Preisbereich
- Testimonials
- FAQ mit aufklappbaren Antworten
- Demo-Kontaktformular

## Hinweis zum Formular

Das Kontaktformular ist in dieser Version nur ein visuelles Demo-Element.  
Es ist noch nicht mit einem echten E-Mail- oder Backend-Service verbunden.

Für eine echte Formularfunktion kann später z. B. eingebunden werden:

- Formspree
- Netlify Forms
- eigenes Backend
